Вкладка «Анализ»

1.	Кнопка «Выбрать пациента». При нажатии открывается окошко со списком пациентов 
        с возможностью поиска по фамилии. 
2.	Поля ФИО и возраст. Заполняются при выборе пациента.
3.	Внизу кнопка «Поставить диагноз». По нажатии открывается окошко с кнопкой ок. 
        Вводим текст в окошко, запись сохраняется в БД в диагноз.

Вкладка «Пациенты»

1.	На всю вкладку список пациентов(ФИО).
2.	Вверху страницы поиск по ФИО.
3.	Рядом с поиском кнопка «+Пациент». При нажатии открывается окошко с полями: 
        Ф, И, О, возраст, паспорт, диагноз, выбрать файл.
4.	Должна быть возможность открыть данные по пациенту на редактирование(открывается окошко как в п.3, 
        внизу кнопку «Подтвердить изменения», «Удалить»)

Нужен тулбар с кнопочками File и Help. По Help открывается окошко где написаны буковки(потом допишу что-н), 
в File хватит выхода.
